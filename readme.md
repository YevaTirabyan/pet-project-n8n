# Test Project for Learning n8n

## Description

This project was created to explore and learn the capabilities of n8n, a workflow automation tool. n8n allows you to build integrations between various services using a visual interface, without the need to write code.

## Project Goals

- Get familiar with the core features of n8n
- Explore the integration of different APIs and services
- Understand how to create automated workflows without programming
- Experiment with creating arguments, variables, and webhooks

## Installation

To work with n8n, you will need Docker. To install and run n8n, follow these steps:

1. Install Docker on your computer if it is not already installed.
2. Start n8n using the following command:

   ```bash
   docker run -it --rm \
     -p 5678:5678 \
     n8n
3. Open your browser and go to http://localhost:5678.

## Usage
1. Create a new workflow by adding nodes to integrate different services.
2. Configure the node parameters and connect them to build your workflow logic.
3. Test and run the workflow, analyzing the results.
